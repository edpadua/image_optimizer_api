## ‚ú® Key Features

The API offers two main endpoints to manipulate and optimize images:

### 1. Format Conversion (`/api/v1/convert`)

* **Purpose:** Converts images to more efficient formats, such as **WebP**, allowing full control over compression quality.
* **Advantage:** Web optimization, reducing file sizes and improving loading times.

### 2. Smart Resizing (`/api/v1/resize`)

* **Purpose:** Resizes images to specific dimensions (`width` and/or `height`).
* **Advantage:** Automatically maintains the **aspect ratio** if only one dimension is provided, preventing distortion. Uses the **Lanczos** filter for the best visual quality during resizing.

## üöÄ How to Run the Project

You can run the API locally via Uvicorn or using Docker.

### 1. Via Uvicorn (Development)

Ensure you have the dependencies installed: `pip install fastapi uvicorn pillow python-multipart`.

Command to run:
uvicorn main:app --reload

The API will be accessible at `http://127.0.0.1:8000`.

### 2. Via Docker (Production)

This method ensures the API runs in an isolated and consistent environment.

1.  **Create the Dockerfile** (Example content):
    FROM python:3.10-slim
    WORKDIR /app
    COPY requirements.txt .
    RUN pip install --no-cache-dir -r requirements.txt
    COPY . .
    CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]

2.  **Build the Image:**
    docker build -t image-optimizer-api .

3.  **Run the Container:**
    docker run -d -p 8000:8000 --name optimizer image-optimizer-api

The API will be accessible via `http://localhost:8000`.

## üí° Usage Examples (cURL)

You can test the endpoints using tools like `cURL` or the `/docs` interface itself.

### 1. Conversion to WebP

Converts an image (`my_image.png`) to `webp` format with quality 80:

curl -X 'POST' \
  'http://127.0.0.1:8000/api/v1/convert?target_format=webp&quality=80' \
  -H 'accept: */*' \
  -H 'Content-Type: multipart/form-data' \
  -F 'file=@my_image.png;type=image/png' \
  --output optimized.webp

### 2. Resizing

Resizes an image to 500 pixels in width (height is calculated automatically):

curl -X 'POST' \
  'http://127.0.0.1:8000/api/v1/resize?width=500&quality=90' \
  -H 'accept: */*' \
  -H 'Content-Type: multipart/form-data' \
  -F 'file=@original_image.jpg;type=image/jpeg' \
  --output resized_500px.jpg
